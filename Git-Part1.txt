Git, Github
--------------
Git - local repository ( source code management system / vesrion control system). 
	Local Repository means that every individual in a team is havaing their own repositories locally

Github - remote repository.
	   The entire team is having one single repository or central repository
 	   At the end of the day, everyone has to push the code in the GitHub repo
	   Remote repository is not available in a local system.This is a web based 
	   application where we have to create an account, commit and push our code

---------
Round 1
-----
1)  create a new local repository  ( one time)

2) provided user info to git repo. (one time)
 
	git config --global user.name "your name"
    	git config --global user.email "your email"


3) Adding files/folders to staging/indexing
   git add -A         -->add all the files and folders to staging
    git add filename
    git add *.java
    git add foldername

4) Commit the code into Local(git) repository
	git commit -m "c ommit message"

5) login to github and create new remote repository.capture url 
connect local repository with remote rep.(One time)
